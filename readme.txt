== Description ==
a plugin for make jquery sliders this is so fast and easy

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php if(function_exists('pars_slider')){ pars_slider(); } ?>` in your templates
